# OP
Implement an operating system-Experiment Homework of 2026 Spring NJU OP Course
## Supported Functions
- builtin commands: exit, cd, env, env-use, env-exit
- run external commands: print extra error information
- redirection
- pipe: any number of pipes can be connected together, but do not support using builtin commands in pipes
- sandbox execution: not complete yet, support basic uses now, but do not support pipe using and execve rules
## Test Files
Mainly used to test sandbox execution. You can edit rules.txt and write in different rules to test the shell.

Due to course requirement, I can not upload test files.
